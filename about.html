About

TrialsManager is an alternative way to share and play tracks on Trials games.

Unfortunately, Ubisoft has been providing minimal support to keep the Trial game servers running.
That's why this exists- to provide an alternative way to download, upload, and manage levels.

Help

Sharing [&] Playing Tracks

This section will show how to manually install levels

Playing Maps/Tracks is easy with the TrialsManager program, but it's perfectly fine if you don't want to use it.

You require TrialsManager to upload maps.

Step 1:
Click on the Map you wish to download on the website.

Step 2:
Download the .tmap file to your corresponding game folder:
"Documents/TrialsFusion/SavedGames"
"Documents/Ubisoft/Trials Evolution Gold Edition/SavedGames".

File Format

A .tmap file is a renamed .zip file containing level data.

Changing the file extension to anything different will make the file unreadable to TrialsManager.
